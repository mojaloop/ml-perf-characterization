   - [YES] Ensure there is a composite index on (`participantId`, `currencyId`,`ledgerAccountTypeId`)

   - [YES] Ensure there is a primary key defined `quoteId`
- **`participantEndpoint` Table:**
   - [YES] Ensure there is a primary key defined `participantEndpointId`
   - [YES] Check there is an index on the `participantId` column
   - [YES] Check if there is an index on the `endpointTypeId` column
   - [NO] Check if there is an index on the `isActive` column
   - [YES] Ensure there is a composite index on (`participantId`, `endpointTypeId`)

- All queries appear to be well-optimized and are efficiently utilizing indexes.
